# Evaluate Learning: Chat Thread Management

## Learning questions

- Are the two swipe directions discoverable and remembered?
- Does Delete confirmation prevent mistakes without feeling heavy?
- Is the short Undo window sufficient for archive, or is an Archived section needed?

## Evidence and decision rule
Use Andrew's simulator/device self-use and any support reports of lost or unrecoverable chats. Proceed as the default if cleanup feels immediate and no accidental loss occurs. Add an Archived section if recovery is wanted after the toast expires; revise the gestures if direction confusion recurs.

## Instrumentation
Rely on existing error reporting and direct interaction observation for this small slice. Do not add behavioral analytics solely to measure private conversation deletion.
