## Frame Recommendation

Run design-thinking-loop with an expanded frame: activity-to-identity Arc creation.

The chip addition is a useful immediate repair, but the larger job is helping Kwilt accept concrete intrinsic motivation as source material and then resolve it into an Arc-worthy identity trajectory. The solution should bend the survey order or add an activity-first lane with an explicit becoming bridge, not merely append more options to the existing adult abstraction ladder.
