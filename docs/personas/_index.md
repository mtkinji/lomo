# Target Audiences And Personas

Target audiences are the market-level groups Kwilt might serve. A persona is a named, concrete representative of one audience. JTBDs describe what becomes urgent for that person in a specific situation.

Use audiences to choose *who* a design is for, personas to make that audience specific, and JTBDs to name the demand that becomes hot for that person.

## Relationship to JTBDs

Audiences, personas, and JTBDs are many-to-many:

- One audience can contain several future personas.
- One named persona can activate several JTBDs depending on the situation.
- One JTBD can appear across several audiences with different stakes and language.
- feature briefs should name the target audience, representative persona, and JTBDs served when the work is large enough to need a design frame.

Each audience also has a **hero JTBD**: the main reason that audience would care about Kwilt. Supporting JTBDs branch off from the hero job, and feature-level JTBDs descend from those supporting jobs.

```text
Target audience
  -> Representative persona
    -> Hero JTBD
      -> Supporting JTBDs
        -> Feature-level JTBDs / sub-jobs
          -> feature briefs / features
```

## Modes

Modes are temporary postures, not audiences:

- Capturing in motion.
- Choosing the few things that matter.
- Making sense of a season.
- Returning after drift.
- Inviting someone in.
- Shaping the system from a command center.

## Using audiences in design work

In `design-thinking-loop` Phase 1:

1. Select the target audience from this index, or create a provisional audience if none fits.
2. Use the named representative persona, adapting only the current situation.
3. Start from that audience's hero JTBD as the main demand hypothesis.
4. Assess which supporting or feature-level JTBDs are active for that persona in this situation.
5. Write the aspirational design challenge from the audience + persona + hero JTBD + active JTBD pairing.

Avoid fluffy archetype labels. Use straight audience language and named representative people.
